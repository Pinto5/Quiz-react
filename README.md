# Which Element Are You? Quiz

A simple personality quiz built with React.  
The user enters their name, answers a few questions, and receives one of four elements: Fire, Water, Earth, or Air.

The result page also displays artwork fetched from the Metropolitan Museum of Art Collection API based on the selected element.

## Features

- User name input with React Context
- Multi-question quiz
- Element result calculation based on selected answers
- React Router navigation
- Artwork fetched from an external API
- Loading and error handling
- Restart quiz button
- Basic responsive styling with CSS

## Tech Stack

- React
- Vite
- React Router
- JavaScript
- CSS
- MET Museum Collection API
